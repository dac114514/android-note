# 极简笔记

Android 笔记应用，基于 Jetpack Compose + Material Design 3 构建。

## 功能

- 富文本笔记编辑（加粗、斜体、标题、列表等）
- 瀑布流笔记列表展示
- 文件夹 + 标签分类管理
- 全文搜索
- 本地 ZIP 备份导出

## 技术栈

- **语言**: Kotlin 100%
- **UI**: Jetpack Compose + Material Design 3
- **架构**: MVVM (Room + ViewModel + Navigation Compose)
- **存储**: Room 本地数据库
- **构建**: Gradle Version Catalog
